新闻速递

逻辑之海：OpenAI模型 disproves discrete geometry conjecture

OpenAI 的模型反驳了离散几何中的一个中心猜想；AI数学研究获重大突破。

OpenAI 宣布其模型通过发现 Erdős 原猜想的反例，成功推翻了该数学命题。这一成果展示了 AI 在跨领域知识迁移方面的能力，将代数数论中的复杂概念引入几何问题。社区讨论集中在模型如何基于现有文献进行微调，以及这种“通过反例证明”相较于正面证明在数学上的意义。有数学家认为，这标志着 AI 正从语言预测工具向真正的科研辅助者转变。

代码黑箱：GitHub确认3800仓库遭恶意VSCode扩展入侵

GitHub确认3800仓库遭恶意VSCode扩展入侵；团队称仅内部数据泄露。

GitHub 确认约 3800 个内部仓库因一名员工安装的恶意 VS Code 扩展而遭入侵。公司已隔离受感染设备并移除了该扩展，评估认为攻击者仅窃取了 GitHub 内部数据。黑客组织 TeamPCP 在暗网论坛声称访问了约 4000 个私有仓库并索要 5 万美元赎金，其行为与此前针对 PyPI、NPM 等供应链的攻击模式一致。此事件再次敲响了开发者工具生态安全性的警钟。

信息迷雾：Google正式在AI搜索结果中引入广告

Google正式在AI搜索结果中引入广告；Gemini生成内容助推广。

Google 正式在 AI 搜索模式中引入两种新广告格式，分别名为“对话式发现广告”和“高亮答案”。这些广告由 Gemini 模型生成，旨在提供相关产品细节和独立解释，以增强用户信任。评论者指出，这是 Google 将广告从网页脚本文本化、以绕过广告拦截器的举措，引发了关于算法中立性与商业变现之间平衡的讨论。

视界重构：逆向工程Apple视频壁纸系统

逆向工程Apple视频壁纸系统；用户可自定义视频桌面。

开发者发布了名为 Phosphene 的开源项目，逆向工程了 Apple 的私有 WallpaperExtensionKit 框架。该工具允许用户将自定义视频文件设置为 macOS 桌面和锁屏壁纸，并支持电源管理和多显示器设置。由于依赖未公开的框架，该功能目前仅支持 macOS 26 (Tahoe) 和 Apple Silicon，但为 macOS 生态系统带来了类似 Windows Vista 的个性化体验。

开源理想：Flipper One发布基于Linux的开放硬件

Flipper One发布基于Linux的开放硬件；专注无闭源驱动。

Flipper One 项目启动，旨在打造一个完全开源的 ARM 计算机。该设备使用主线 Linux 内核，不含闭源驱动或二进制 blob，旨在解决当前 ARM Linux 生态中 BSP（板级支持包）封闭的痛点。团队将其与现有的 Flipper Zero 区分开来，后者专注于低层硬件访问，而 Flipper One 则专注于 IP 网络、SDR 和本地 AI 等基于 Linux 的任务。

终端新解：Rmux提供Playwright风格的终端复用器

Rmux提供Playwright风格的终端复用器；支持脚本化与多会话。

Rmux 是一个用 Rust 编写的可编程终端复用器，提供类似 Playwright 的 SDK。它兼容 tmux 命令，允许开发者通过代码脚本检查和驱动终端会话，旨在解决长期运行代理进程的会话管理问题。该项目目前处于公开预览阶段，支持 Linux、macOS 和 Windows，试图在保持 tmux 兼容性的同时，为自动化工具提供更友好的接口。

琴弦之争：Fender对S型吉他发起大规模版权诉讼

Fender对S型吉他发起大规模版权诉讼；小厂商面临生存危机。

Fender 已向美国吉他制造商 LsL Instruments 发出停止侵权通知，并据称已向数十家其他公司发出了类似信函。该行动紧随 Fender 在德国赢得的一项保护 Stratocaster 外形设计的判例之后，该公司声称任何在欧盟销售此类产品的行为均构成侵权。小企业面临被迫召回并销毁现有库存的威胁，引发了关于行业创新与公司知识产权保护之间界限的讨论。

历史余烬：1945年核试验图像修复面世

1945年核试验图像修复面世；摄影记录首次完整公开。

IEEE Spectrum 报道，一组 1945 年三位一体核试验的图像已获修复并公开。这些照片由 Berlyn Brixner 拍摄，展示了爆炸初期阶段，捕捉到了当时肉眼无法看到的细节。历史学家称，尽管 52 台相机中只有 11 台成功工作，但这些影像为理解核爆炸的规模提供了关键证据，是物理学史上的珍贵资料。

编辑手记

OpenAI 模型对离散几何猜想的反驳是一个极具象征意义的信号，它表明 AI 技术正从“统计预测”向“逻辑推理”与“知识发现”深水区迈进。这一突破不仅验证了大型模型在处理抽象数学问题上的潜力，也引发了学术界关于 AI 辅助科研新范式的讨论。然而，这种能力的边界依然存在争议，目前的材料显示，AI 更擅长在已有知识库中进行“微调”与发现反例，而非构建全新的理论框架。

与此同时，科技行业的商业逻辑与安全挑战依然严峻。Google 在 AI 搜索中引入广告，清晰地展示了技术进步如何服务于核心变现模式；而 GitHub 遭遇的供应链攻击则暴露了即使是巨头，也无法完全规避开发者工具生态中的信任风险。这两者共同指向了一个现实：技术越先进，其背后的生态构建与安全防护就越脆弱。

从硬件领域看，Flipper One 的发布与 Fender 的版权围剿形成了有趣的对比。前者代表了对开放硬件与自由软件的极致追求，试图打破 ARM 生态的闭源垄断；后者则展现了传统制造业巨头利用法律手段固化市场优势的趋势。这种“开源理想”与“商业围剿”的拉锯，或许才是当前科技生态中最值得观察的长期博弈。

接下来值得观察的具体信号包括：OpenAI 的数学突破是否能被其他团队在更短时间内复现，以及 Flipper One 的开源策略能否真正推动 ARM 主线内核的普及；此外，Google 的新广告格式在用户中的实际接受度，以及 Fender 的诉讼是否会在欧盟引发针对外观专利保护的广泛反弹。
