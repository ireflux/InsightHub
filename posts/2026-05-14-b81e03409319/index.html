Linux gaming 因内核优化迎来新速度；iPhone 芯片驱动廉价 MacBook Neo；Anthropic 推出面向小企业的 Claude；美国地方域名免费注册流程公布；经典游戏 Scorched Earth 推出 Web 版；Windows 7 风格的 Windows 10 模组发布。

新闻速递

《内核里的 Windows：Linux 游戏性能的跃迁》

Linux gaming is faster because Windows APIs are becoming Linux kernel features

2026 年 3 月，Linux 在 Steam 平台的用户份额首次突破 5%，创下历史新高。Windows 10 结束支持以及 Steam Deck 的普及，推动了这一操作系统在桌面端的广泛采用。过去 Linux 游戏性能的提升主要依赖于 Wine 和 Proton 这类翻译层的软件优化，而最新的进展显示，核心性能的提升正在向更底层的 Linux 内核迁移。NTSYNC 是一个新增的内核级驱动程序，它为 Linux 提供了原生实现 Windows 特定同步工具的能力，使得 Wine 不再需要模拟这些机制。Wine 11 版本对内核运行机制进行了重写，NTSYNC 默认加载于所有更新的 Steam Deck 上，带来了巨大的性能增益。

《iPhone 芯片的笔记本：廉价 Mac 的极限挑战》

MacBook Neo Deep Dive: Benchmarks, Wafer Economics, and the 8GB Gamble

Apple 于 2026 年 3 月发布了 MacBook Neo，这是该品牌历史上最便宜的笔记本，起售价为 599 美元。该机型搭载了与 iPhone 16 Pro 相同的 A18 Pro 芯片，基于 TSMC 第二代 3nm 工艺制造。Geekbench 6 单核得分约为 3569，介于 M3 和 M4 芯片之间；但在全负载运行约 60 秒后，受限于无风扇散热设计，CPU 利用率会降至 64%。该机型仅提供 8GB 内存和 256GB SSD 的配置。评论指出，8GB 内存可能导致 macOS 占用超过 75% 的内存空间，存在引发计划报废法规诉讼的风险，且其 I/O 接口仅支持 10Gb/s 的 USB，缺乏 Thunderbolt 支持。

《工作流里的 Claude：企业 AI 的落地尝试》

Claude for Small Business

Anthropic 推出了名为“Claude for Small Business”的服务，旨在将 AI 嵌入小企业日常依赖的工具中。该服务通过 Claude Cowork 平台，将 Claude 集成至 QuickBooks、PayPal、HubSpot、Canva 等工具中，提供 15 个预置的自动化工作流，涵盖薪资规划、月度结账、销售营销和发票催收等场景。用户需在 Claude 中激活相关功能并批准操作后，系统才会执行发薪、发布或付款等动作。虽然官方强调安全性与可审批性，但社区讨论集中在潜在的“恐怖故事”上，担忧 AI 可能出现幻觉导致税务错误或资产被没收。

《被遗忘的互联网角落：美国地方域名的注册指南》

Setting up a free *.city.state.us locality domain (2025)

美国存在一种免费的地方域名系统，格式为 *.city.state.us，最早创建于 1992 年。注册此类域名需要美国公民身份或永久居留权，且必须通过 Amazon Lightsail 获取名称服务器。用户需向当地政府的委托管理者提交《临时 .US 域名模板》来完成注册。评论指出，如果该域名对应的注册商已倒闭，注册将变得极为困难；此外，该域名系统禁止使用 WHOIS 隐私服务，这对个人隐私构成潜在威胁。

《像素风暴中的坦克：Web 版 Scorched Earth 2000》

Scorched Earth 2000 – Web

经典回合制射击游戏《Scorched Earth》的爱好者团队发布了一款基于 Web 的移植版，名为 Scorched Earth 2000，最新版本为 v1.1（2026 年 5 月）。该版本完全使用 JavaScript 编写，保留了原版的核心玩法、多人联机功能和自定义武器系统。社区评论普遍将其视为童年的怀旧之作，部分用户分享了通过修改游戏文件实现“作弊”的早期黑客经历。

《时光倒流：Windows 7 的幽灵在 Windows 10 中苏醒》

Classic 7 is a Windows 10 LTSC mod to look 1:1 to Windows 7

名为 Classic 7 的项目利用 Windows 10 IoT Enterprise LTSC 2021 系统修改而来，旨在提供 1:1 的 Windows 7 界面体验。该项目重现了 Aero Glass 效果、桌面小部件以及 Windows Media Center 等功能。由于兼容性限制，部分 3D 动画效果和系统组件无法恢复。用户评论反映出对 Windows 7 界面的偏好，但也指出该系统无法恢复 Windows 7 时代的启动速度与硬件效率。

编辑手记

Linux gaming 的加速与 MacBook Neo 的推出，揭示了两个截然不同但同样激进的技术演进路径。前者（Linux）通过接纳 Windows 的 API 设计并将其写入内核，试图抹平系统间的体验鸿沟；后者（Mac）则通过将移动端芯片塞入笔记本并限制内存规格，在极端成本控制下挑战硬件的物理极限。这两种尝试都在压缩用户的选择空间：Linux 正变得“更像 Windows”，而 Mac 则试图在廉价与高性能之间寻找新的平衡点。

AI 在企业级应用中的落地同样面临信任危机。Anthropic 将 Claude 深度嵌入财务与业务工具，虽然展示了自动化流程的潜力，但社区关于“幻觉导致灾难性后果”的担忧并非空穴来风。当 AI 从“助手”转变为“执行者”时，其输出结果的不可验证性成为了商业应用的最大障碍。

此外，技术社区对“旧事物”的执着也值得玩味。从 Scorched Earth 的 Web 移植到 Classic 7 的界面复刻，这些项目往往不是为了创造新的技术奇迹，而是为了修复某种体验上的遗憾。这暗示了在快速迭代的科技产业中，人们对于稳定、熟悉界面的渴望依然存在。

接下来值得观察的具体信号是：Linux 在 Steam 上的份额能否在下一个财年突破 10%；Apple 的 8GB 内存限制是否会在欧盟或美国引发实质性的反垄断或消费者保护诉讼；以及 Anthropic 的企业版 AI 在实际运行中是否会频繁触发“审批”机制，从而暴露其当前架构的局限性。
