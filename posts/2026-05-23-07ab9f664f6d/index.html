新闻速递

蹲坑与芯片：Toto的跨界野心Why Japanese companies do so many different things

Toto 不仅是全球最大的马桶制造商，其陶瓷部门自 1988 年起便生产半导体制造所需的静电吸盘。随着 AI 对高带宽内存需求的激增，这一部门已成为公司利润的主要来源。这一案例揭示了日本企业传统多元化策略在当前技术周期下的新价值，几乎所有日本公司都像 Toto 一样涉足千差万别的领域。

虚拟红队：AI安全的新防线Project Glasswing: An Initial Update

Anthropic 推出 Project Glasswing，利用 Claude Mythos Preview 扫描全球关键软件。合作伙伴在一个月内发现了超过一万起高危漏洞，Mozilla 在测试中修复了 271 个漏洞。英国 AI 安全研究所报告称 Mythos Preview 是首个解决其多步网络攻击模拟的模型。目前的挑战已从“发现漏洞”转变为“验证与修补”。

Copilot 的回归：微软的AI工具博弈Microsoft starts canceling Claude Code licenses

微软曾向数千名开发者提供 Anthropic 的 Claude Code 工具，但发现开发者更倾向于使用 GitHub Copilot CLI。微软计划于 6 月 30 日取消大部分 Claude Code 许可证，转而全面支持 Copilot CLI。这一决定被解读为成本控制与产品战略的双重考量，微软也停止了收购 Cursor 的计划。

开源契约的裂痕：Bambu争议BambuStudio has been violating PrusaSlicer AGPL license

Prusa 创始人 Josef Prusa 指控 Bambu Studio 在分 fork PrusaSlicer 后一直违反 AGPL-3.0 协议，特别是保留了专有的网络二进制文件。他认为这违背了开源精神，并指出 3D 打印在中国已成为战略产业。评论指出，中国法律环境可能迫使公司做出此类选择，而开源协议的执行成本高昂且困难。

跨越边界：寄往难民营的笔记本Shipping a laptop to a refugee camp in Uganda

作者试图向乌干达难民营的朋友寄送笔记本电脑，却遭遇了复杂的国际物流障碍。澳大利亚邮政因锂电池问题拒绝空运，改用货运代理后费用翻倍。作者不得不预付关税缓冲金，并面临漫长的清关等待。这一过程反映了跨国物流中政府收费、海关流程以及灰色市场运输的现实困境。

编辑手记

Toto 的案例表明，传统制造业的积累可以转化为高科技供应链的关键力量。这种跨行业的“隐性知识”积累是许多日本企业的核心竞争力，也是西方企业难以复制的护城河。

微软在内部工具上的选择，反映了科技巨头在 AI 生态中的博弈。放弃外部工具而转向自研，既能控制数据流，也能在财务周期末削减开支。这种“用脚投票”的选择往往比战略规划更真实地反映了技术落地的实际需求。

Anthropic 的结果显示，AI 在网络安全领域的应用已从实验阶段进入实战阶段，但“发现-修补”的效率差距仍是最大挑战。目前的瓶颈不在于模型能力，而在于全球软件供应链的修复速度。

Bambu Studio 的争议则触及了开源协议在复杂地缘政治环境下的脆弱性。如果开源项目无法提供足够的经济或法律保护，商业公司可能会选择规避协议。这不仅是法律问题，更是地缘政治对企业行为约束的体现。
